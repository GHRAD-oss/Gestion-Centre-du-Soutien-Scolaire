Instructions d'installation

  1. Téléchargez le fichier DemoCentreSoutien.zip ci-dessus
  2. Extrayez l'archive dans un dossier de votre choix (ex: C:\GestionCentreSoutien)
  3. Exécutez le fichier Setup.exe en tant qu'administrateur
  4. Suivez les instructions de l'installateur
  5. Lancez l'application depuis le menu Démarrer ou le raccourci sur le bureau
  6. Utilisez les comptes de démonstration fournis :
    • Administrateur : admin / admin123
    • Utilisateur standard : user / user123

Configuration système requise

Minimum :
  • 2 GB RAM
  • 500 MB espace disque
  • Windows 10/11
  • .NET 6.0 Runtime
Recommandé :
  • 4 GB RAM
  • 1 GB espace disque
  • Windows 10/11 64-bit
  • .NET 8.0 Runtime
